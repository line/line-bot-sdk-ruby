# Echo Bot

A simple bot application that uses the LINE Messaging API SDK for Ruby to return text messages sent by a user.

## Usage

```bash
$ export LINE_CHANNEL_SECRET=YOUR_CHANNEL_SECRET
$ bundle install
$ bundle exec ruby app.rb
```
